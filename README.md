Overview
This Django application allows users to check the current weather for any city. 
Users can input a city name, and the app fetches real-time weather data including temperature, 
weather condition, and an icon representing the weather. 
The app also displays a background image of the city using Google Custom Search API.

Features
-Search weather by city name
-Displays temperature, description, and weather icon
-Dynamic background image corresponding to the city
-Responsive design for desktop and mobile
-Error handling for invalid city inputs

Technologies Used
-Python 3.x
-Django 4.x
-HTML5, CSS3
-Requests library
-OpenWeatherMap API
-Google Custom Search API
